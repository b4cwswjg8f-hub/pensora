Wie wird berechnet?

Pensora zeigt alle Formeln offen — keine Blackbox, kein Marketing-Optimismus. Jede Zahl ist auf geprüfte gesetzliche Grundlagen und öffentlich zugängliche Daten zurückführbar. Autor: Niall Bradfield, unabhängiger Finanzberater, Stuttgart.

Beamtenpension

Rechtsgrundlage: § 14 BeamtVG (Bund) · § 6, § 9, § 12a BeamtVG · Landesbeamtenversorgungsgesetze
1,79375 %
Ruhegehaltssatz pro Jahr
71,75 %
Maximaler Ruhegehaltssatz
35 %
Mindestversorgung

Schritt 1 — Ruhegehaltfähige Dienstjahre

// Effektive ruhegehaltfähige Dienstjahre (gdj) Reguläre Dienstjahre = aktuelles Jahr − Diensteintritt − Beurlaubungszeit Zukünftige Jahre = Pensionsalter − aktuelles Alter Ohne Teilzeit: gdj = Reguläre Dienstjahre + Zukünftige Jahre Mit Teilzeit (tzPct % über tzJ Jahre): Vollzeitjahre = Reguläre Dienstjahre − tzJ gdj = Vollzeitjahre + tzJ × (tzPct / 100) + Zukünftige Jahre
Hinweis Elternzeit: Elternzeit bis 3 Jahre wird vollständig als ruhegehaltfähige Dienstzeit angerechnet (§ 6 Abs. 1 Satz 2 BeamtVG). Beurlaubung ohne Dienstbezüge wird dagegen abgezogen.

Schritt 2 — Ruhegehaltssatz und Bruttopension

// Ruhegehaltssatz (§ 14 Abs. 1 BeamtVG) satz = min(gdj × 0.0179375, 0.7175) = min(gdj × 1,79375 %, 71,75 %) // Endbesoldung (inflationiert bis Pensionseintritt) eg = Pensionsbesoldung × (1 + Steigerung/100)^Restjahre // Bruttopension brutto = max(eg × satz, Mindestversorgung)

Schritt 3 — Steuer

// Versorgungsfreibetrag 2025: 14,4 % des Jahresbetrags, max. 1.080 €/Jahr // Sinkt jährlich bis 2040 auf 0 % vfb = min(brutto × 12 × 0.144, 1080) // Werbungskostenpauschale: 102 €/Jahr zvE = max(brutto × 12 − vfb − 102, 0) // Einkommensteuer → progressiver Tarif (§ 32a EStG, siehe Abschnitt 07) Steuer/Monat = taxDE(zvE, Kirchensteuerpflicht) / 12

Schritt 4 — KV/PV und Nettopension

// GKV-Versorgungsbezüge: 8,3 % der Bruttopension kv_GKV = brutto × 0.083 // PKV mit Beihilfe: eigener Beitrag × (1 − Beihilfesatz/100) kv_PKV = PKV-Monatsbeitrag × (1 − beihilfe/100) // Pflegeversicherung: 3,4 % der Bruttopension pv = brutto × 0.034 Nettopension = brutto − Steuer/Monat − kv − pv

Gesetzliche Rente

Rechtsgrundlage: § 64 SGB VI · § 70 SGB VI · DRV Rentenwert 2025
40,79 €
Rentenwert 2025 (ab Juli)
50.493 €
Durchschnittsentgelt 2025
96.600 €
Beitragsbemessungsgrenze 2025

Entgeltpunkte und Bruttorente

// Entgeltpunkte pro Jahr (§ 70 SGB VI) ep = min(Bruttolohn, 96.600) / 50.493 // Gesamtentgeltpunkte (Vergangenheit + Zukunft), max. 1,95 EP/Jahr gesEP = min(ep × (ArbeitsjahreVerg + RestjahreBisRente), Gesamtjahre × 1.95) // Rentenwert inflationiert bis Renteneintritt (Rentenanpassungsrate) rvJetzt = 40.79 × (1 + Rentenanpassung/100)^Restjahre // Bruttorente (§ 64 SGB VI: EP × Zugangsfaktor × Rentenartfaktor × Rentenwert) // Zugangsfaktor = 1.0 bei Regelalter, Rentenartfaktor = 1.0 bei Altersrente brutto = gesEP × 1.0 × 1.0 × rvJetzt

Besteuerung und Nettorente

// Besteuerungsanteil 2025: 83,5 % — steigt bis 2040 auf 100 % bestAnt = min(100, 83.5 + (Rentenjahr − 2025) × 0.5) / 100 // Werbungskostenpauschale Rente: 102 €/Jahr zvE = max(brutto × 12 × bestAnt − 102, 0) // KV: 8,3 %, PV: 3,4 % der Bruttorente Nettorente = brutto − taxDE(zvE)/12 − brutto × 0.117 // Kaufkraft (reale Nettorente nach Inflation) nettoReal = Nettorente / (1 + Inflation/100)^Restjahre

Sparbuch-Vergleich

Zinseszins-Endkapital

// Monatliche Rendite r = Jahresrendite / 12 // Laufzeit in Monaten n = Laufzeit_Jahre × 12 // Endwert aus monatlichen Einzahlungen (Annuität fällig) fvSparplan = Monatsbetrag × ((1+r)^n − 1) / r × (1+r) // Endwert des Startkapitals fvStartkapital = Startkapital × (1+r)^n // Gesamtendkapital fvGesamt = fvSparplan + fvStartkapital // Reales Endkapital nach Inflation fvReal = fvGesamt / (1 + Inflation/100)^Laufzeit_Jahre

Entnahme nach der 4 %-Regel

// Nachhaltige Jahresentnahme (4 %-Regel, Bengen 1994) Entnahme_Jahr = fvGesamt × 0.04 Entnahme_Monat = Entnahme_Jahr / 12 // Zeitlich begrenzte Entnahme (exakte Rentenformel) eExakt = fvGesamt × r / (1 − (1+r)^−nEntnahme) // wobei nEntnahme = Entnahmedauer in Monaten // Benötigtes Kapital für Ziel-Entnahme (25er-Multiplikator) kapitalZiel = ZielEntnahme_Monat × 12 / 0.04
Sparbuch-Vergleich: Pensora berechnet denselben Sparplan mit 0,5 % p.a. (Sparbuch/Tagesgeld), um den Zinseszins-Vorteil des ETF-Depots zu quantifizieren.

Einkommensteuer (§ 32a EStG)

Rechtsgrundlage: § 32a EStG · Solidaritätszuschlag § 3 SolZG · Kirchensteuer 8–9 % je Bundesland

Pensora berechnet die Einkommensteuer nach dem progressiven Tarif 2025. Alle Rechnungen verwenden das zu versteuernde Einkommen (zvE) nach Abzug aller spezifischen Freibeträge des jeweiligen Produkts.

// § 32a EStG — Einkommensteuertarif 2025 (Single) // Zonen (gerundet): zvE ≤ 11.784 €: Steuer = 0 // Grundfreibetrag 11.785 ≤ zvE ≤ 17.005 €: Progressionszone 1 // 14–24 % 17.006 ≤ zvE ≤ 66.760 €: Progressionszone 2 // 24–42 % 66.761 ≤ zvE ≤ 277.825 €: Spitzensteuersatz // 42 % zvE > 277.825 €: Reichensteuersatz // 45 % // Solidaritätszuschlag (seit 2021 nur für höhere Einkommen) SolZ = max(0, min(ESt × 0.055, (ESt − 17.543) × 0.119)) // Kirchensteuer (falls anwendbar) KiSt = ESt × 0.09 // Bayern/BW: 8 %, sonst 9 %
Splitting-Vorteil: Verheiratete Paare können das Ehegattensplitting (§ 26b EStG) nutzen — das halbiert rechnerisch das zvE und reduziert die effektive Steuerlast erheblich. Pensora berechnet aktuell für Singles; für Paare empfehlen wir ein persönliches Beratungsgespräch.
